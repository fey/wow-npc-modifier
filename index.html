Инструкция

  1. Перейти на WoWHead и найти страницу нужного NPC
  2. Скопировать из адресной строки ID непися.
  3. Выбрать уровень, класс, моба, желаемое количество HP
  4. Нажать "Отправить"
  5. Скопировать запрос.




    
UPDATE creature_template
SET HealthModifier = 5.0505
WHERE entry = @тут_айди_моба_с_вовхеда;
